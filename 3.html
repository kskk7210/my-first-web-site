JavaScript

JavaSript is a cross-flatform,object-oriented scripting language developed by Netscape was first used in Netscape browsers. However, Netscape was working with European Computer Manufacturers Association (ECMA) to deliver a standardized, international programming language based on core JavaScript.

ECMA-262 or the ECMAScript Language Specification defines the ECMAScript Language, or just ECMAScript.
